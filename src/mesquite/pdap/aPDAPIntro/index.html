PDAP:PDTREE: A translation of the PDTREE module of Garland et al.'s Phenotypic Diversity Analysis Programs

Peter E. Midford, Theodore Garland Jr., and Wayne Maddison

August 2009

This package of modules and library classes was brought over from version 5.0 of PDAP.  PDAP was originally described in Garland, Dickerman, Janis, and Jones (1993), but several functions pertaining to reconstructed root values were added later (see Garland, T., Jr., P. E. Midford, and A. R. Ives. 1999. An introduction to phylogenetically based statistical methods, with a new method for confidence intervals on ancestral values. American Zoologist 39:374-388.; Garland, T., Jr., and A. R. Ives. 2000. Using the past to predict the present: confidence intervals for regression equations in phylogenetic comparative methods. American Naturalist 155:346-364. and the documentation for PDAP, PDINSTRW.DOC).  Current functionality includes what is in PDTREE, i.e., computation of independent contrasts, diagnostic screens to check adequacy of branch lengths, transformations of branch lengths, and some conventional statistical analyses. Other components of PDAP, such as simulation functionality, will be added soon. News and the latest update for this package are available here.

Overview

The translation of PDAP currently includes:
  • Import and Export of .PDI files, including name conversion as needed.
  • Exporting of Nexus tree and character information to .PDI files.
  • Dignostic Screens and Statistics These are available once a PDAP diagnostic window is opened.
    • Absolute values of standardized contrasts vs. their standard deviation
    • Absolute values of standardized contrasts vs. the (corrected) height of their base node
    • Absolute values of standardized contrasts vs. the estimated character value at their base node
    • (Corrected) height of internal nodes vs. their estimated character values
    • Contrast vs. contrast plots
    • Contrast residuals vs. X value and base node height
    • Statistics pertaining to reconstructed root values (Garland, Midford, and Ives, 1999).
    • Generation of .FIC files. These have been tested with recent versions of Microsoft Excel.
  • Branch length transformations and estimates These are available in the tree window.
    • Log base 10 transform of branch lengths
    • Exponential transform of branch lengths
    • Transform branch lengths by Grafen's (1989) rho
    • Set all branch lengths to a constant (1.0)
    • Set branch lengths according to a method of Grafen (1989)
    • Set branch lengths according to a method of Pagel (1992)
    • Set branch lengths according to a method suggested by S. Nee (cited in Purvis 1995)
  • Character (tip) data manipulations These are available in the character matrix editor.
    • Log base 10 transform character values
    • Base 10 antilog transform character values
    • Exponential transform of character values
    • Raise character values to a power
    • Divide one character value by another
  • Export of distance matrix files equivalent to the DOS PDDIST application
    • (1)Pairwise differences of tip data values from character matrix (for a specified character)
    • (2)Patristic distance
    • (3)Node count along path between tips (speciation patristic)
    • (4)Shared proportional history distance
    • (5)Shared history distances. This is the DSC matrix of PDDIST (C matrix of Garland and Ives 2000), i.e., the phylogenetic variance-covariance matrix as would be used for PGLS analyses.

Installation

The compressed file you download will contain a directory (folder) called pdap_mes. This will, in turn, contain two directories, one titled PDAP and one titled examples. To install the PDAP modules, the "PDAP" directory must be installed in the "mesquite" directory within the directory titled "Mesquite_Folder".

Examples

A demo which includes examples based on the 49lbr and appamg91 trees distributed with DOS PDAP is included. The examples include a navigation window that allows the user to scroll through the examples in order. The example set starts with the file Part1.nex.  There are additional examples including an updated version of the 49lbr tree with full species and character names.
